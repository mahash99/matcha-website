# Maison Matcha Website

A premium one-page static website for a matcha store.  
The website is designed to feel feminine, elegant, wellness-focused, trustworthy, and conversion-focused.

It is built using only:

- HTML
- CSS
- JavaScript

No React, Next.js, backend, database, or paid tools are required.

---

## Website Sections

The website includes:

1. Header / Navigation
2. Hero Section
3. Problem / Desire Section
4. Benefits Section
5. Product Section
6. How to Prepare Section
7. Reviews Section
8. Brand Story Section
9. Matcha vs Coffee Comparison
10. FAQ Accordion
11. Final Call-to-Action
12. Footer
